# SPARKEL (SPARql Knowledge Engine Layer)

This is a piece of software that can act as a generic bridge between a SPARQL endpoint and the REST API of a smart connector.

## Only reactive
Since it wraps a SPARQL endpoint, it can only offer reactive knowledge interactions.
For now, only ANSWER interactions.

## Configuration (future work)

For now, we will hard-code the knowledege interactions.
In the future we could read them from a configuration file.
